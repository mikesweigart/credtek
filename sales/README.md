# CredTek Sales Pack

Sales infrastructure for the first 50 conversations. Edit freely — this is
your team's living playbook, not a doc set in stone.

## Files

- **[cold-outbound.md](./cold-outbound.md)** — email templates (cold to
  Director of Credentialing, cold to PE-backed COO, warm intro follow-up,
  no-response follow-up) + LinkedIn DMs + sending cadence
- **[demo-talk-track.md](./demo-talk-track.md)** — minute-by-minute 20-min
  demo script with screenshot cue points, objection responses, and post-call
  email template
